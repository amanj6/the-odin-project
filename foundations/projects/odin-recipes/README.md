# Odin Recipes

This is my first project from the Foundations course of The Odin Project.  
The goal of the project was to build a simple recipe website using only HTML.

This project focuses on mastering **HTML fundamentals**, **Git workflows**, and **project structuring** by building a multi-page recipe website.

---

## What this project does

The website includes:

- A homepage with links to recipes
- Separate pages for each recipe
- Ingredients lists
- Step-by-step cooking instructions

This project is intentionally simple and focuses on learning the basics of HTML and how websites are structured.

---

## What I practiced

While making this project I practiced:

- Writing basic HTML
- Using headings, paragraphs, lists, and links
- Organizing files into folders
- Linking multiple pages together
- Using Git and GitHub to track changes
